# 视图模式切换

Grid 支持在表格、卡片、列表三种视图模式之间切换。该功能默认关闭，需要显式调用启用。

## 基本使用

```php
// 启用表格和卡片两种模式，默认显示表格
$grid->viewMode('table', ['table', 'card']);
```

启用后，工具栏会出现视图切换按钮组，用户点击即可切换视图模式。

## 三种视图模式

### 表格模式 (table)

默认的数据表格视图，与不启用 `viewMode` 时完全一致。

### 卡片模式 (card)

以响应式卡片网格展示数据，每行最多 4 列（大屏 4 列、中屏 3 列、小屏 2 列、手机 1 列）。第一列作为卡片标题，其余列作为卡片内容行。

### 列表模式 (list)

以紧凑的列表形式展示数据，每条记录一行，字段水平排列。

## 配置示例

```php
// 三种模式全部启用
$grid->viewMode('table', ['table', 'card', 'list']);

// 仅启用表格和列表
$grid->viewMode('table', ['table', 'list']);

// 默认显示卡片模式
$grid->viewMode('card', ['table', 'card']);
```

## 工作原理

- 视图模式通过 URL 参数 `_view_` 传递，如 `?_view_=card`
- 切换模式不影响筛选条件、分页、排序等功能
- 所有模式共享相同的数据层（Model、Filter），只替换展示层
- 不启用 `viewMode()` 时，Grid 行为完全不变
