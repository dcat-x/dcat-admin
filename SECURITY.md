# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

如果你发现安全漏洞，请**不要**公开提交 Issue。

请通过以下方式报告：

1. 发送邮件至 myxiaoao@gmail.com
2. 在邮件中详细描述漏洞信息

我们会在 48 小时内回复确认收到报告，并在 7 天内提供修复方案或进展更新。

## 安全更新

安全更新会通过以下方式发布：

- GitHub Releases
- Packagist 版本更新
- CHANGELOG.md 更新

建议用户保持 Dcat Admin X 更新到最新版本。
